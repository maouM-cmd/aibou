# デモ動画 台本（3分以内）

画面録画ソフト（Windows標準: `Win + G` でXbox Game Bar → 録画ボタン）を起動してから、
このコマンドを順番にターミナルへコピペしてください。ナレーションはそのまま読み上げるか、
声が不要なら字幕・テロップでもOKです（Devpostの規約上どちらでも問題ありません）。

事前確認: `knowledge/preferences/coding_style.md` に「実務でClineを使用中」という記録があり、
下記の質問文で確実にこの記憶がヒットすることをテスト済みです。

## 0:00–0:25（25秒）コンセプト説明

**ナレーション**:
> 「これはAIBOU、私が学びや好みを蓄積し続けている個人用の相棒AIです。Ollamaでローカル動作するツールとして、日常的に使っています。今回、このAIBOUの永続記憶機能をQwen Cloudでも動かせるように対応しました。」

**画面**: `README.md`を開いて冒頭を見せる、またはターミナルで
```bash
cd C:\Users\admin\.gemini\antigravity\scratch\aibou
dir
```

---

## 0:25–1:15（50秒）ローカル(Ollama)でのメモリ実演

**ナレーション**:
> 「まずローカル環境で、AIBOUに聞いてみます。『私が実務で使っているAIコーディングツールは何？』」

**コマンド**:
```bash
python ask.py "私が実務で使っているAIコーディングツールは何？"
```

**期待される応答**: 「Cline」という単語を含む回答（`preferences/coding_style.md`の記録を参照）

**ナレーション**:
> 「ちゃんと過去に記録した好みを覚えていて、実務でClineを使っていると答えてくれました。これがChromaDBによる永続記憶です。質問のたびに関連する過去の経験をベクトル検索して、文脈に組み込んでいます。」

---

## 1:15–2:15（60秒）Qwen Cloudへの切り替えデモ

**ナレーション**:
> 「次に、同じ知識ベースをQwen Cloud経由で使ってみます。環境変数を1つ変えるだけです。」

**コマンド**（Windows PowerShellの場合）:
```powershell
$env:AIBOU_PROVIDER = "qwen"
python ask.py "Clineを使いこなすコツを教えて"
```

**期待される応答**: `.clinerules`やPlan→Actワークフローなど、`knowledge/tips/cline_tips.md`の内容を踏まえた回答

**ナレーション**:
> 「バックエンドをOllamaからQwen Cloudに切り替えても、同じ記憶を参照して答えてくれています。呼び出し先はaibou.pyの_call_qwen_rawメソッドで、Alibaba CloudのDashScope互換APIを直接叩いています。」

**画面**: `aibou.py`の`_call_qwen_raw`関数を少し見せる（エディタで開いておく）
